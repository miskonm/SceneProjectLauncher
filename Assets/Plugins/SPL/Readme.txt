SPL - Scene Project Launcher

GET STARTED 

- After importing a SPL the required asset with settings will be created in the folder ../SPL/Resources/SPLSettings
- To change settings go "Tools -> SPL -> Settings"
- Choose needed option and create enum with scenes info. For it press button 'Recreate Scene Enum'. It will add all selected scenes in your Build Settings. After compile select the scene you want and have fun.
